This study is going to take ~20 minutes.

You are going to read Chinese characters and respond to them. On each trial, you will first see two characters, one after the other, for example: 花 and 草. Then, you will see one of these characters again (that is, either 花 or 草). Your task is to decide whether the last character is the same as the first or the second one. That is:

  • If you see 花 - 草 - 花

    • The last character matches the first one
    Please press 1 to indicate that it matches the first character

  • If you see 花 - 草 - 草

    • The last character matches the second one
    Please press 2 to indicate that it matches the second character

Remember that the last character will always be the same as either the first or the second character you saw. Please respond as quickly as you can while ensuring accuracy!
